# Unity Souls Like Combat System
## Content:
 - Complex state machine and behaviour tree design.
 - Reuse of code between AI and player.
 - Physics based projectiles.
 - Weapons with custom combos.
 - Abilities.
 - Dodging.
 - Targeting.
 - Knockback & Knockdown.
